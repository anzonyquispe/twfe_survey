# Archivos excluidos del repositorio

Estos archivos superan el límite de 2 GB de GitHub LFS y no pueden subirse al repositorio.

## 1. Diamond et al. (2019) — `usa_00012.dta` (3.3 GB)

- **Ruta original:** `data/2015-2019/Diamond et al. (2019)/data/AER-2018-1289.R2_replication/data/census/usa_00012.dta`
- **Motivo de exclusión:** 3.3 GB, supera el límite de 2 GB por archivo de GitHub LFS.
- **Descripción:** Microdatos del censo de EE.UU. (IPUMS USA extract).
- **Cómo obtenerlo:**
  1. Registrarse en [IPUMS USA](https://usa.ipums.org/usa/) (acceso gratuito).
  2. Crear un extract con las variables y muestras utilizadas en Diamond et al. (2019).
  3. Alternativamente, los archivos de replicación del paper están disponibles en el [AER Data Editor](https://www.openicpsr.org/openicpsr/project/116446/version/V1/view) (openICPSR #116446).
  4. Colocar el archivo descargado en: `data/2015-2019/Diamond et al. (2019)/data/AER-2018-1289.R2_replication/data/census/`

*Nota: Todos los demás archivos binarios grandes (100 MB - 2 GB) se gestionan mediante Git LFS.*
