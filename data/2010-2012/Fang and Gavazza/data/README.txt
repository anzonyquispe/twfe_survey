The files included are:

1- Stata do files. These files are:

tables2_3_5.do 
tables2_3_5col4.do 
table4.do
table6.do
table7.do
firsms4.do
hrs_instr1.do
hrs_newinst.do

The files whose name starts with "table" replicate the tables in the paper. 
The other files are ancillary files called by the "table" files.

2- Stata datasets:

ads-laws.dta (from David Autor website. This file constains the instruments used in Table 7)
statesector.dta (from CBP. This file contains the instruments used in Section IV) 
dyn_st_89_98s1.dta (from CBP. This file contains the instruments used in Section IV) 
gdpdefl.dta (GDP deflator)
rndhrs_h.dta (HRS dataset, RAND version)
the folders meps and BHPS contain MEPS and BHPS datasets.

To Replicate the results:
Copy all files in a directory, change the names of the local directories in each file, and run each file.
