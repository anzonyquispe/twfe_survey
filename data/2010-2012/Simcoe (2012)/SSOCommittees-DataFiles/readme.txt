This directory contains all the files needed to create the Tables in "Standard Setting Committees: The Governance of Shared Technology Platforms" by Timothy Simcoe. Please cite that paper if you use these data or programs. To reproduce the Tables and Figures from the paper (and a few more), open Stata and cd to this directory, then run the "analysis.do" program, and examine the log files in the ./tables sub-directory. 

All of the data used in the analysis are located in the "data/idLevel.dta" file. These data are documented in the file "idLevel-Doc.txt". The programs sub-directory contains do-files that perform the analysis. The results are saved in the tables sub-directory.

I have included a few extra data sets in the data sub-directory: 

1) The files email_panel_*.out are comma-delimited ASCII text files that contain raw counts of email messages sent to IETF Working Group email listservs. These files were used to create the Suit-Share and Message count variables in the paper. The file "data/email_panel.out" contains counts of all messages sent to WG's in my sample. The file "data/email_panel_replies.out" screens out spam and other random messages by only counting Reply messages (i.e. threads). The file "data/email_panel_gt4.out" focuses on active WG participants, defined as users who submit 4 or more messages to the WG listserv, and is used to create the variables used in the paper.

2) The file "idVersionLevel.dta" contains submission dates for each revision of every Internet Draft in the data. Thus, where idLevel.dta is a cross-section of Internet Drafts, this file contains additional longitudinal data on the revision history of each ID. To be clear, I refer to a series of documents with a common filename (e.g. draft-simcoe-badidea-NN.txt) as an Internet Draft. Within the IETF, it is common to refer to each version of the same document (e.g. draft-ietf-simcoe-badidea-01.txt versus draft-ietf-simcoe-badidea-02.txt) as an Internet Draft.
