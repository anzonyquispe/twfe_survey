Sub-directories are as follows/gis: Contains results of GIS software spatial merges and do-files that process these data sets/data: Contains a series of sub-directories each with different input data sets.  Data sets used for empirical analysis	are saved in this directory/	/ccdb: City county data book data	/cen90: Some stf3 1990 census data (extracted from Geolytics)	/cenpan: 1960, 1970, 1980 and 1990 stf4 census data	/check: alternative tracts 1960 data set used to check allocation assumptions	/district70: Census district data from 1970	/district80: Census district data from 1980	/district90: Census district data from 1990	/handenter: Hand entered data from the 1960 census	/ipums: 1960 census micro data	/welch_ccd: Common core of data on district student composition and Welch/Light desegregation data	/xwalk: Data sets and do-files that generate crosswalks between various geographies and coding schemes/programs: Contains two important do-files that build the data/tables: Contains the do-files that create output for tables and figuresTo build the data sets, the do-files should be run in the following order/gis/cln-seggis-all.do: Allocates tracts in all years to school district geographies using unclipped tracts/gis/cln-seggisx.do: Allocates tracts in all years to school district geographies using tracts clipped to 1970 tracted areas/programs/build-districts.do: Builds a district level data set with information on desegregation and student demographics/programs/panel-censusx.do: Combines resulting data sets from the above three do-files into 	tractpanx.dta: used for spatially disaggregated analysis within central districts	dis70panx.dta: which is used for district/time level analysisThe do-files in the tables directory are named to match the tables and figures in the paper.  Some of them generate .out 	files which have regression output.
