This readme.txt file describes the sources for the datasets in
data/cenpan for the school desegregation project.

nhgist60.dta: Census tract data from 1960 downloaded from nhgis.org.
nhgisc60.dta: Census county data from 1960 downloaded from nhgis.org.
tracts70.dta: Census tract data from 1970 read in from the stf4a files.
cnty70.dta: Census county data from 1970 read in from the stf4c files.
tracts80.dta: Census tract data from 1980 read in from the stf4a files.
cnty80.dta: Census county data from 1980 read in from the stf4b files.
tracts90.dta: Census tract data from 1990 read in from the stf4a files.
cntys90.dta: Census county data from 1980 read in from the stf4c files.
