This folder contains the files needed to carry out the simulations analysis in the manuscript and
the GIS basin-wise work (undertaken in response to referees, but not included in the final paper; 
however, some programs did maintain a dependency on some of the files produced here, so this
process has been included.)

Instructions--

Simulations: in the matlab folder, one needs to adjust filepaths and run the make_aco_matlab4.sh script. 

GIS basins work: in the GIS folder, one needs to adjust filepaths and run findwatershed2.py in ARCGIS and
 then assemble the results in Stata using assemble_watersheds3.do.
